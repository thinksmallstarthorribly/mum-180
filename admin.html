How This Works

Codes live in two places: This admin page (your records) and access.html (the actual validation). When you add a code here it saves to your browser's localStorage as your record — but you also need to add it to the VALID_CODES array in access.html on GitHub.

To add a code to access.html: Go to access.html on GitHub → click the pencil icon to edit → find the VALID_CODES array → add your new code → commit.

To revoke a code: Mark it revoked here (your records), then remove it from the VALID_CODES array in access.html on GitHub. The client's existing session lasts up to 7 days after their last login.
